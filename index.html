Q.1 You are given a string S, and your task is to return an array B(having a size of 2), where B[0] contains the count of character A (uppercase) in string S and B[1] contains the count of character D (uppercase) in string S. Note: You have to complete countCharacters function. No need to take any input.

Ans.1 const countCharacters = (S) => 
        {
          var B=[];
          var counta=0;
          var countd=0;
           for (i=0; i < S.length;i++)
           {
             if (S.charAt(i)==="A")
             {
               counta++;
             }
             if (S.charAt(i)==="D")
             {
               countd++;
             }
           }
           B[0]=counta;
           B[1]=countd;
           return B;
        };

Q.5 You are given a string S consisting of two letters A and D,where each character represent the winner of N games played between Aditya and Danish, where letter A represents the win of Aditya and letter D represents the win of Danish. Ans.5 var Game_Winner = (S) => 
        {
          var countA=0;
          var countD=0;
          for (i=0; i < S.length; i++)
          {
            if (S.charAt(i)==='A')
            {
              countA++;
            }
            else
            {
              countD++;
            }
          }
          if (countA>countD)
          {
            return "Aditya";
          }
          else if (countD>countA)
          {
            return "Danish";
          }
          else 
          {
            return "Draw";
          }
        }

Q.11 Tina is given a string S which contains the first letter of all the student names in her class. She got a curiosity to check how many people have their names starting from the same alphabet. So given a string S , she decided to write a code that finds out the count of characters that occur more than once in the string.Note: You have to complete Count Head function. No need to take any input.

Ans.11 var Count_Occ = (s) => 
      {
         var arr=s.split("");
         var filterarr=arr.filter((a,b) => arr.indexOf(a)!==arr.lastIndexOf(a));
         filterarr.sort();
         var S="";
         var count=1;
         for (i=0; i < filterarr.length; i++)
         { 
           if (filterarr[i]===filterarr[i+1])
            {
              count+=1;
            }
            else 
            {
              S+=filterarr[i] + count;
              count=1;
            }
          }
          return S;
      };
